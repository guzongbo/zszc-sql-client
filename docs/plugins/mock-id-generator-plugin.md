# 随机信息与密码工具插件

`随机信息与密码工具插件` 是 `zszc-sql-client` 当前内置的组合型工具插件，定位为测试造数与账号辅助工具工作区。

## 插件定位

- 面向测试数据准备与账号辅助场景
- 把多种常用生成能力集中到一个插件工作区内
- 适合开发、自测、联调和演示时快速生成样本数据

## 主要功能

### 随机信息生成

- 支持生成姓名
- 支持生成统一社会信用代码
- 支持生成身份证号
- 支持生成银行卡号
- 支持生成手机号、邮箱、住址

### 密码生成

- 支持长度配置
- 支持大小写、数字、符号开关
- 支持最少数字数与最少符号数
- 支持避免易混淆字符

### 密码短语生成

- 支持单词数配置
- 支持分隔符配置
- 支持首字母大写
- 支持追加数字

### 用户名生成

- 支持长度配置
- 支持风格选择
- 支持分隔符配置
- 支持数字后缀
- 支持避免易混淆字符

## 典型使用场景

- 生成测试账号、测试密码和演示数据
- 为接口联调、自动化测试或数据库初始化准备样本内容
- 快速得到更接近真实格式的数据，而不是手工临时拼接

## 技术形态

- 前端：React
- 后端：Rust
- 通信方式：`stdio JSON-RPC`
