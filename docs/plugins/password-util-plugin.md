# 密码工具插件

`密码工具插件` 是 `zszc-sql-client` 当前内置的插件样板之一，定位为密码处理工具插件，同时也承担插件工作区集成样例的作用。

## 插件定位

- 面向密码处理场景的轻量工具插件
- 用于演示宿主如何加载插件前端页面
- 用于演示插件前端如何通过桥接层调用 Rust 后端
- 用于演示插件后端如何以 `stdio JSON-RPC` 方式响应宿主请求

## 主要功能

### 密码解密

- 接收待处理密文
- 支持按现有工具约定处理下划线替换场景
- 返回解密后的明文结果

### 数据库存储密文生成

- 接收用户输入的明文密码
- 生成可用于数据库存储的密文结果
- 适合密码处理、配置准备或测试辅助场景

## 典型使用场景

- 需要对既有密文做快速解密校验
- 需要生成数据库可用的密码密文
- 需要参考一个最小但完整的插件实现样板

## 技术形态

- 前端：React
- 后端：Rust
- 通信方式：`stdio JSON-RPC`
